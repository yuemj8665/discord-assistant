### 1. launchd PATH 오류 수정

**증상**: `claude CLI가 설치되지 않았거나 PATH에 없습니다.`
**원인**: launchd가 쉘 PATH를 상속받지 않음
**해결**: plist에 `EnvironmentVariables.PATH` 추가 (`/Users/mamyeongjae/.local/bin` 포함)
**결과**: claude CLI 정상 인식 ✅

---

### 2. Claude CLI 응답 시간 초과 수정

**증상**: `Claude CLI 응답 시간이 초과되었습니다.`
**원인 1**: `subprocess.run()`이 asyncio 이벤트 루프를 블로킹 → Discord heartbeat 끊김
**해결 1**: `loop.run_in_executor(None, self._llm.ask, user_input)` 적용
**원인 2**: `stdin` 미설정으로 claude CLI가 입력 대기
**해결 2**: `stdin=subprocess.DEVNULL` 추가
**원인 3**: timeout 60초 → 무거운 작업(폴더 분석 등) 초과
**해결 3**: `timeout=300` (5분)으로 증가
**결과**: 일반 메시지 정상 응답, 무거운 작업도 처리 ✅
